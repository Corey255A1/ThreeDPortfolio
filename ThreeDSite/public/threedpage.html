Edscape Editor

Edscape is a school focused escape room editor and controller. Written in C# UWP for the frontend GUI and a shared .NetStandard 2.0 dll for some of the backend. It uses a very basic custom HTTP webserver since UWP doesn't ship with one. This is so that the rooms can be hosted and controlled directly off the teachers computer without the need for the internet.

Edscape Local Preview Controller

The Edscape Local Preview is an application written in C# WPF as a way to locally run the Edscape developed rooms on the same laptop. UWP has the limitation of being restricted from connecting to LocalHost servers. This application imports the Edscape room packages and runs them locally.

Project Time Keeper

Github Source!

Time Keeper is a C# WPF application to track the time spent on different projects in real time. Add in the project charge numbers for easy tracking. Activities can be exported to a CSV file

Darwin Core KML Utility

Github Source!

This application can import data from a Darwin Core style CSV and export it out in Google Earth KML. Writen in C# WPF. It allows the user to customize the Google Earth Placecard and visualize the result in realtime. The data can also be configured to be grouped into folders based on the headers.
